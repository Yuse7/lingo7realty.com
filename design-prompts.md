# Готовые промты для claude.ai/design — копируй по одному блоку

Каждый блок ниже — **самодостаточный промт** (бренд-токены и правила языка уже внутри). Копируй **один блок целиком** → вставляй в claude.ai/design → получаешь один экран. Готовый HTML кидай мне — подгоню под бренд и вставлю в секцию.

**Правила языка (вшиты в каждый промт, для справки):**
- Интерфейс (контролы, кнопки, счётчики) — **на английском** (Save question, Next, Question 4/20, Audio…).
- Названия разделов/тем — **параллельно EN→RU** (EN сверху, RU мелким снизу).
- Учебный контент (вопросы, варианты, разборы, текст) — **параллельно EN→RU**.
- Перевод помечается бейджем «RU» (не глобусом). Добавление слова в L7 Dictionary — по тапу на слово (всплывает мини-меню). В шапке — чип языка «RU». Русские пояснительные подписи внутрь экрана не вставляем (они уйдут в текст лендинга).
